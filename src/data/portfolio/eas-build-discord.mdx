## Overview

`EAS Build Webhook Notification` is a serverless lambda to notify the result of EAS build.

## Objectives

- Primary objective with this project was to learn build automation
- Wanted to create pipelines in AWS for faster mobile builds
- Also got exposure to serverless, lambda functions for AWS
- Discord API is fun to experiment with!
