> `useLayoutEffect` is similiar to `useEffect`, see the difference [here](https://kentcdodds.com/blog/useeffect-vs-uselayouteffect)

```js
export default function LovelyComponent() {
  const mounted = useRef(false)

  const doSomething = useCallback(() => {
    // do something
  }, [])

  useLayoutEffect(() => {
    if (!mounted.current) {
      mounted.current = true
      return
    }

    doSomething()
  }, [doSomething])

  return (
    <>
      <p>Little ice cream nuggets!</p>
    </>
  )
}
```
