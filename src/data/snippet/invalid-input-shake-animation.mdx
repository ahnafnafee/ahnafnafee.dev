```html
<input type="text" placeholder="Enter username" pattern="[a-z]*" />
```

```css
@keyframes shake {
  25% {
    transform: translateX(0.5rem);
  }
  50% {
    transform: translateX(-0.5rem);
  }
  75% {
    transform: translateX(0.5rem);
  }
}

input:invalid {
  animation: shake 0.3s;
  /* other style */
}
```
